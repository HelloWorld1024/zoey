Zoey, mobile UI framework

What is Zoey?

Zoey is a mobile framework on top on zepto.js that aims to be small and fast. The golder rule is: don't generate any extra markup. Do everything through progressive enhancements and CSS3.

Why?

Size matters. At least when it comes to mobile devices. Zoey tries to keep it simple and weights at only 6KB JavaScript + 16KB CSS (without GZip).

To improve the download size even further, zepto.js is used instead of the full-blown desktop version of jQuery.

Styling

You want your application to be unique. Zoey uses Compass to allow you to easily customise the default theme colours.

By overriding predefined variables, you can produce a unique theme of your own. You can even go further and completely re-design the default one, it's entirely up to you.

Platforms

Zoey has been tested on modern mobile browsers and works on Android 2.2+ and iOS 4.2+. Earlier versions might be supported as well, but haven't been tested.

The default theme makes use of CSS3 goodies: gradients, box sizing, shadows, etc. Some of these may not be available in previous Android/iOS builds.

Help & How-to

Since this is an early release, documentation has not been finalised yet.

Poke around the source code as well to find out what's supported.

Contribute

Zoey is very much a work in progress (hence no official release yet).

You can help in a lot of ways: adding documentation, writing tests, performing code reviews, testing on different OSes and versions, opening issues, creating new themes and anything you reckon will benefit the project in general.
