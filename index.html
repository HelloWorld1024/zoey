Zoey

What is it?

Zoey is a lightweight framework for developing mobile apps. It is small, customisable and features a range of UI components.

Overview

Zoey is built on top of zepto.js and weights at only 21.6KB (combined JavaScript and CSS) and GZipp'ed at around 6KB.

Zoey is designed for HTML5 and supports modern browsers (iOS 4.2 / Android 2.2 tested). Using progressive enhancements and CSS3, your DOM is left untouched and no extra markup is added to the document.
