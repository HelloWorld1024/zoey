Your First App

Zoey mobile apps are HTML5 websites. For the purpose of this tutorial, the relevant lines from Usage will be omitted.

Let's begin by creating an empty HTML5 website:

<!doctype html>
<html lang=en>
<body>
</body>
</html>

Every app consist of a series of pages (e.g., a log in screen). To create a page:

<!doctype html>
<html lang=en>
<body>
<article data-role=page>
</article>
</body>
</html>

Zoey relies on data-* attributes to create UI components and attach behaviour. data-role is a special attribute assigning a particular role to an element of the DOM.

The Basics

Let's go ahead and add a header and a footer to our app:

<!doctype html>
<html lang=en>
<body>
<article data-role=page>
<header data-role=header>
</header>
<footer data-role=footer>
</footer>
</article>
</body>
</html>
